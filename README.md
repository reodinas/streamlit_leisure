# Menu
- Home: 데이터와 데이터의 출처, 메뉴에 대한 설명을 나타냅니다.
- Chart: 원하는 데이터를 보기 쉽게 차트로 시각화 합니다.
- admin: 관리자 컨셉으로 만든 페이지 입니다.  매주 추가 조사해 제공되는 데이터를 지속적으로 관리할 수 있습니다.
